# GitHub Actions Assignment

This repository was created for a university assignment to demonstrate the use of
GitHub Actions workflows.

## Repository Purpose
The purpose of this repository is to show how a GitHub Actions workflow can be
attached to a repository and automatically executed when changes are pushed to
the main branch.

## GitHub Actions Workflow
The workflow in this repository runs on every push to the `main` branch and
performs the following actions:
- Checks out the repository contents
- Lists the files in the repository
- Prints a confirmation message to indicate successful execution
